# Student Performance Analysis and Exam Preparation Schedule Planner

## Description
A Python-based data analytics project that analyzes student performance, attendance, and study habits. It identifies weak subjects, calculates rankings, and generates personalized exam preparation schedules.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib

## Features
- Student performance analysis
- Average marks calculation
- Ranking system
- Weak subject detection
- Study schedule generation
- Performance visualization

## How to Run

Install requirements:

pip install pandas numpy matplotlib

Run:

python student_analysis.py
